MS Internet Explorer 6 or Earlier Detected!

Details:

Q-Cogo currently does not support early versions of Microsoft Internet Explorer. To use Q-Cogo, you MUST upgrade to a newer version browser.


Why doesn't Q-Cogo work in Internet Explorer 6 or earlier?
Q-Cogo does not yet support Internet Explorer versions 6 or earlier due to limitations with these browser versions. Early MS browsers did everything differently, making it hard to write the same software for everyone. The decision was made to block Internet Explorer 6 users entirely rather than lead them to a half-functional application.

Will Q-Cogo ever work in Internet Explorer 6 or earlier?
Maybe. Making web applications work with outdated browsers is a lot of work, and that time might be better spent adding more features for everyone to enjoy. Future versions of Q-Cogo MAY become more backward-compatible.

What can I do about this?
Download a newer version of Internet Explorer, or, preferrably, a more reliable browser like Mozilla Firefox. In addition to being able to use great web applications like Q-Cogo, your computer will be safer and will work more quickly online.
